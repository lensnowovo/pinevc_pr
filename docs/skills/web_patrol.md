# Skill: 网页巡逻 (web_patrol)

> 参谋官核心技能

## 触发条件

- 定时触发 (每日凌晨 1:00-6:00)
- 手动触发 (用户要求 "巡逻一下")

## 信息源列表

### 官方公告
```
优先级: P0 (最高)

• NMPA 药监局
  URL: https://www.nmpa.gov.cn/

• CDE 药品审评中心
  URL: https://www.cde.org.cn/

• FDA (美国)
  URL: https://www.fda.gov/

• 医保局
  URL: http://www.nhsa.gov.cn/
```

### 行业媒体
```
优先级: P1

• 动脉网
  URL: https://www.vbdata.cn/
  类型: 综合医疗健康

• 医药魔方
  URL: https://www.pharmcube.com/
  类型: 医药数据

• 36氪 (医疗频道)
  URL: https://36kr.com/medicine

  类型: 前沿技术

```
优先级: P2

  说明: 关注的行业大号

## 执行步骤

### Step 1: 按优先级访问信息源
```
1. 先访问 P0 官方公告
2. 再访问 P1 行业媒体
3. 最后浏览 P2 社交媒体
```

### Step 2: 收集最新内容
```
时间范围: 最近 24 小时

收集字段:
• 标题
• 来源
• 发布时间
• 链接
• 摘要 (前 200 字)
```

### Step 3: 初步筛选
```
排除条件:
• 与医疗健康无关
• 纯广告/软文
• 重复内容
• 负面敏感信息

保留条件:
• 与松禾投资领域相关
• 与 CDE+AI 论点相关
• 涉及被投企业
• 重大行业事件
```

### Step 4: 评分排序
```
评分维度 (各 1-10 分):

1. 相关性: 与松禾业务的相关程度
2. 时效性: 新闻的新鲜程度
3. 影响力: 事件的重要程度
4. 可创作性: 内容丰富度

总分 = 相关性×0.4 + 时效性×0.2 + 影响力×0.2 + 可创作性×0.2
```

### Step 5: 生成巡逻报告
```
格式:

# 🔍 巡逻报告 | {日期}

## ⭐ Top 5 热点

### 1. {标题}
- 来源: {来源}
- 时间: {时间}
- 评分: {分数}
- 摘要: {摘要}
- 链接: {URL}
- 推荐理由: {一句话}

### 2. ...

---

## 📊 今日数据
- 巡逻信息源: {数量}
- 收集内容: {数量}
- 筛选后: {数量}

## 💡 建议
{是否值得跟进创作}
```

## 输出格式

发送巡逻报告给大管家。

## 注意事项

1. **频率控制**: 不要过于频繁访问同一网站
2. **来源可信**: 优先官方来源，社交媒体需交叉验证
3. **避敏**: 涉及敏感话题要标注
