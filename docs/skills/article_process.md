# Skill: 文章加工 (article_process)

> 笔杆子核心技能

## 触发条件

当大管家分发文章链接时，或用户直接发送文章请求加工时触发。

## 执行步骤

### Step 1: 获取文章内容
```
1. 使用浏览器访问文章链接
2. 提取标题、正文、作者、发布时间
3. 识别文章类型 (新闻/深度/观点)
```

### Step 2: 提取核心信息
```
1. 总结文章核心事件 (100字内)
2. 提取 3-5 个关键数据点
3. 识别涉及的企业/机构
4. 识别涉及的领域/赛道
```

### Step 3: 匹配论点框架
```
查询知识库，匹配 CDE+AI 论点:

• Clinical (临床转化)
  关键词: IND、临床试验、突破性疗法、CDE...

• Discovery (创新来源)
  关键词: 靶点发现、新模态、First-in-class、AI药物设计...

• Engineering (工程验证)
  关键词: CMC、规模化生产、质量控制...

• AI (智能加速)
  关键词: AI药物发现、生成式AI、AlphaFold...

输出: 匹配的论点 ID 和相关性评分
```

### Step 4: 匹配被投企业
```
查询被投企业库:
1. 赛道匹配 (文章领域 = 企业赛道)
2. 论点匹配 (文章论点 ∩ 企业论点标签)
3. 时效性 (企业近期有相关进展)

输出: 推荐引用的企业列表 (最多3家)
```

### Step 5: 生成内容
```
按模板生成公众号文章:

# {标题}

## 核心要点
- 要点 1
- 要点 2
- 要点 3

## 事件背景
{100-200字客观描述}

## 行业影响
{关联核心论点的分析，200-300字}

## 松禾视角
{我们的观点，关联 CDE+AI}

## 我们的布局
{引用被投企业案例}

---
标签: #标签1 #标签2 #标签3
```

## 输出格式

```markdown
[完整公众号文章，Markdown 格式]
```

## 质量检查清单

- [ ] 标题吸引眼球但不夸张 (≤20字)
- [ ] 关联了至少1个核心论点
- [ ] 引用了至少1家被投企业 (如有相关)
- [ ] 无禁用词 (参考 blacklist.md)
- [ ] 语调符合品牌规范
- [ ] 字数在 800-1500 范围

## 注意事项

1. **保密**: 不泄露未公开信息
2. **准确**: 数据需有来源，不确定的要标注
3. **专业**: 医药术语使用准确
4. **品牌**: 保持松禾的专业形象
