# Skill: 早报生成 (daily_brief)

> 早报官核心技能

## 触发条件

- 定时触发 (每日 8:00)
- 手动触发 (用户要求 "生成早报")

## 执行步骤

### Step 1: 收集隔夜新闻

```
来源:
1. 参谋官的巡逻结果 (热点池)
2. 官方公告 (NMPA/FDA)
3. 行业媒体头条

时间范围: 过去 12-24 小时
```

### Step 2: 筛选 Top 5

```
筛选标准:
1. 重要性: 对行业的影响程度
2. 相关性: 与松禾投资领域的关联
3. 时效性: 越新越好
4. 可读性: 适合早报格式

数量: 5 条 (最多不超过 7 条)
```

### Step 3: 生成早报内容

```
按模板生成:

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

☀️ 松禾医健 · 今日大事
{日期} | 第 {N} 期

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📊 昨夜今晨，{数量} 条行业要闻

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1️⃣ {标题}
{50字摘要}
💡 松禾快评: {一句话点评}

2️⃣ {标题}
{50字摘要}
💡 松禾快评: {一句话点评}

... (共 5 条)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🎯 今日关注
{一条深度话题推荐，100字内}

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🔗 详情链接
• {标题}: {URL}
• {标题}: {URL}
...

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

松禾资本医健团队
专业 · 深度 · 前瞻
```

### Step 4: 质量检查

```
检查项:
□ 标题不夸张
□ 摘要简洁准确
□ 点评有观点但不极端
□ 无敏感信息
□ 格式整齐
```

### Step 5: 用户确认

```
发送预览给用户:

"早报已生成，请确认是否推送:
{早报预览}

[确认推送] / [需要修改]"
```

### Step 6: 推送

```
用户确认后:
1. 推送到微信群 (如有)
2. 推送到企业微信 (如有)
3. 记录推送状态
```

## 早报风格指南

### 语调
```
✅ 专业但不枯燥
✅ 简洁但信息量够
✅ 有观点但不过激

❌ 不八卦
❌ 不标题党
❌ 不造谣
```

### 点评风格
```
好的点评:
• "国内首个，关注后续临床进展"
• "与上月 FDA 批准的 XX 形成呼应"
• "验证了我们关于 CDE 效率的判断"

不好的点评:
• "重磅！" (太空)
• "太厉害了" (不专业)
• "碾压竞品" (过激)
```

## 输出格式

完整的早报文本，可直接复制发送。

## 注意事项

1. **时间敏感**: 早报必须在 8:30 前完成
2. **信息准确**: 数据要有来源
3. **观点适度**: 点评要有但不极端
4. **格式统一**: 每天格式保持一致
