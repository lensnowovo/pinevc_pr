# OpenClaw (龙虾/元气AI) 获取指南

## 简介

OpenClaw（中文名"龙虾"，产品名"元气AI"）是一款能够掌控电脑、模拟人类操作的 AI Agent 平台。

核心特点：
- 🖥️ **电脑全权限** - 能操作浏览器、软件、文件
- 🧠 **多层记忆系统** - 身份记忆 + 技能记忆 + 工作流程
- ⏰ **持续执行** - Heartbeat 心跳 + Cron 定时
- 🎓 **自主学习** - 能自己读文档、学技能
- 💬 **自然语言交互** - 不需要编程

### 第一次对话示例

```
你: 你好，我想让你帮我处理一些工作

AI: 你好！我是你的 AI 助手。我可以帮你做什么？

你: 我需要你帮我处理老板转发的文章，生成公众号内容

AI: 好的，我可以帮你做这个。请告诉我：
    1. 你希望文章的风格是什么？
    2. 有没有需要特别关注的品牌要求？

你: (继续对话，开始训练...)
```

## 训练心法

### 1. 把 AI 当新人养

```
❌ 错误心态: 期望 AI 一次就完美
✅ 正确心态: 接受试错，越错越准

AI 犯错 → 你纠正 → AI 记录为 Skill → 下次不错
```

### 2. 用自然语言说话

```
❌ 错误: 写代码让 AI 执行
✅ 正确: 像跟员工说话一样

示例:
"这篇文章的重点是 xxx，请帮我提炼 3 个核心观点"
"这个风格不对，要更专业一点，像我们之前那篇 xxx"
```

### 3. 多互动，多反馈

```
傅盛 14 天发了 44 万字指令

高频交互 = AI 越懂你 = 效率越高
```

### 4. 建立规则，持续迭代

```
保密规则示例:
"发送任何信息前，先问自己 4 个问题：
 1. 是否老板授权？
 2. 是否涉及隐私？
 3. 是否让老板尴尬？
 4. 老板知道会不会生气？"
```

## 常见问题

### Q: OpenClaw 和 ChatGPT/Claude 有什么区别？

```
ChatGPT/Claude: 对话机器人，只能聊天
OpenClaw: AI 员工，能操作电脑、记住经验、主动工作
```

### Q: 需要编程基础吗？

```
不需要。全程自然语言交互。
```

### Q: 数据安全吗？

```
• 本地运行，数据在自己电脑
• 敏感文件可设置访问权限
• 建议初期就建立保密规则
```

### Q: 可以多台电脑同步吗？

```
• Session 记录在本地
• Skill 文件可导出导入
• 可通过云盘同步
```
