## 语调指南

### 整体风格
- **专业但不生硬**: 展现深度，但保持可读性
- **前瞻但不夸张**: 有观点，但有依据
- **自信但不傲慢**: 展示判断力，但不贬低他人

### 正面示例

```
✅ "松禾早期布局的康宁杰瑞，其双抗平台已进入 III 期临床，印证了我们对中国创新药临床转化能力的判断。"

✅ "中国临床效率比美国快 12-18 个月，这正在成为全球药企的新选择。"

✅ "从 fast-follower 到 first-in-class，中国创新药正在升级。"
```

### 负面示例

```
❌ "中国医药完爆美国" - 过于绝对，缺乏依据

❌ "碾压式领先" - 不专业，易引发争议

❌ "松禾精准抄底" - 投资术语不适合对外传播

❌ "XX 公司股价将翻倍" - 涉及股价预测，合规风险
```

## 禁用词列表

### 完全禁用

| 类别 | 禁用词 | 原因 |
|------|--------|------|
| 夸张词 | 完爆、碾压、秒杀、吊打 | 不专业 |
| 绝对词 | 100%、一定、绝对、必然 | 过于绝对 |
| 炒作词 | 史诗级、震撼、惊天 | 标题党 |
| 敏感词 | 内幕、独家爆料 | 合规风险 |

### 谨慎使用

| 词汇 | 替代建议 |
|------|----------|
| 领先 | "处于第一梯队" / "具有竞争力" |
| 独家 | "少数几家" / "早期布局" |
| 最大 | "规模领先" / "重要参与者" |
| 首家 | "首批" / "先行者" |

## 内容结构规范

### 标题规范

```
✅ 好标题:
- "ADC 药物出海：中国创新药的下一站"
- "从康宁杰瑞看双抗赛道的临床效率"
- "AI + 制药：重塑药物发现的 12 个月"

❌ 差标题:
- "震惊！中国 ADC 完爆美国"
- "松禾投资必看：XX 公司将翻 10 倍"
- "重磅消息：XX 获批"
```

### 正文结构

```markdown
# 标题

## 核心要点
- 3-5 个要点，每个 20 字以内

## 事件背景
100-200 字，客观描述

## 行业影响
200-300 字，行业分析

## 松禾视角
200-300 字，结合论点

## 我们的布局
100-200 字，企业案例

---
标签：#标签1 #标签2 #标签3
```

### 篇幅要求
- **最佳篇幅**: 800-1500 字
- **最短**: 不低于 600 字
- **最长**: 不超过 2000 字

## 引用规范

### 企业引用格式

```
✅ 正确格式:
"松禾早期布局的{企业简称}，其{核心产品/技术}已进入{阶段}，印证了我们在{论点维度}的判断。"

✅ 实例:
"松禾早期布局的康宁杰瑞，其 PD-L1/CTLA-3 双抗已进入 III 期临床，印证了我们对中国创新药临床转化能力的判断。"
```

### 数据引用规范

```
✅ 有来源:
"据 CDE 公布的数据，2025 年 IND 申请数量同比增长 35%。"
"根据公司公告，该药物从 IND 到临床仅用 8 个月。"

❌ 无来源:
"据说..."
"据业内人士透露..."
"市场传言..."
```
