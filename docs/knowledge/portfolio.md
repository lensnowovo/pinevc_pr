## 使用说明

本文档存储松禾医健团队被投企业的关键信息，用于：
1. 内容生成时自动关联相关企业
2. 提供引用案例的准确信息
3. 保持品牌输出的准确性

**请根据实际情况填写企业信息**

## 企业信息模板

### 企业填写指南

每家企业需填写以下字段：

| 字段 | 说明 | 必填 |
|------|------|------|
| id | 唯一标识 | ✅ |
| name | 企业全称 | ✅ |
| short_name | 简称 (用于文章引用) | ✅ |
| description | 一句话描述 | ✅ |
| thesis_relevance | 论点相关性 (1-3) | ✅ |
| sectors | 赛道标签 | ✅ |
| stage | 当前阶段 | ✅ |
| highlights | 亮点 (用于引用) | ✅ |
| recent_news | 近期动态 | ⬜ |
| quote_template | 引用话术模板 | ⬜ |

- **重大事件**: 及时更新 (融资、获批、合作等)
- **年度复盘**: 重新评估论点相关性

### 数据来源
- 企业官方公告
