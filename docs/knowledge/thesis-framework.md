# 年度论点框架 (2026)

> 核心战略文档 | 最后更新: 2026-03-01

## 年度主题

**CDE + AI**

> 中国创新药的竞争力 = Discovery × Clinical × Engineering
> AI 是每个环节的加速器

---

## 核心论点详解

### C - Clinical (临床转化)

**核心观点**: 中国的临床效率是核心竞争优势

**关键词**:
- IND、临床试验、患者入组
- 突破性疗法、加速审批、CDE

**叙事要点**:
1. 中国临床效率比美国快 12-18 个月
2. 集中化试验站点 + 大患者池
3. CDE 审批效率持续提升
4. 全球 First-in-Human 首选中国

**引用话术**:
> "在 Clinical 环节，中国的临床试验效率正成为全球药企的新选择。松禾布局的 {企业}，其 {项目} 从 IND 到临床仅用 {时间}，印证了中国临床加速的判断。"

---

### D - Discovery (创新来源)

**核心观点**: 从 fast-follower 到 first-in-class 的转变

**关键词**:
- 靶点发现、新模态、First-in-class
- AI 药物设计、分子生成、创新药

**叙事要点**:
1. 中国创新药从 me-too 向 me-better / first-in-class 升级
2. 新模态 (ADC、双抗、CGT) 领域全球领先
3. AI 加速早期发现，缩短 50% 时间
4. 专利数量全球领先

**引用话术**:
> "在 Discovery 环节，中国创新药正在从 fast-follower 转向 first-in-class。松禾投资的 {企业}，其 {项目} 是真正的 First-in-class，代表了中国创新的新高度。"

---

### E - Engineering (工程验证)

**核心观点**: CMC 能力决定商业化成败

**关键词**:
- CMC、规模化生产、质量控制
- 成本优势、供应链、GMP

**叙事要点**:
1. CMC 是从临床到商业化的关键桥梁
2. 中国制造的成本优势显著
3. 质量标准与国际接轨
4. 供应链自主可控

**引用话术**:
> "在 Engineering 环节，CMC 能力决定了创新药能否真正惠及患者。{企业} 的 {项目} 展现了中国药企在工艺开发上的成熟度。"

---

### AI (智能加速)

**核心观点**: AI 正在重塑药物发现全流程

**关键词**:
- AI 药物发现、深度学习、生成式 AI
- AlphaFold、蛋白质设计、分子生成

**叙事要点**:
1. AI 加速靶点发现和验证
2. 生成式 AI 用于分子设计
3. AI 优化临床试验设计
4. 中国 AI 制药专利全球领先 (6x 美国)

**引用话术**:
> "AI 正在重塑药物发现全流程。松禾在 AI 制药领域布局的 {企业}，正在用 AI 加速 {环节}，将传统需要 {时间} 的工作缩短到 {更短时间}。"

## 论点组合策略

### 单论点文章
适用于事件只涉及单一维度

```
结构: 事件 → 单一论点深度 → 企业案例 → 结论
示例: 某企业 IND 获批 → Clinical 效率 → 被投案例
```

### 双论点文章
适用于事件涉及两个维度

```
结构: 事件 → 论点1 → 论点2 → 企业案例 → 结论
示例: 某企业 ADC 进入临床 → Discovery (新模态) + Clinical (效率)
```

### 全论点文章
适用于重大事件或年度总结

```
结构: 事件 → CDE 三维度 → AI 加速 → 多企业案例 → 展望
示例: 年度盘点、重大政策解读
```

## 禁用/谨慎使用的叙事

### 禁用
- "中国医药超越美国" (过于绝对)
- "完爆"、"碾压" (不专业)
- 具体股价、市值预测 (合规风险)

### 谨慎使用
- 与竞品直接对比 (需客观有据)
- 疗效数据引用 (需注明来源)
- 未来预测 (需加 "预计"、"可能" 等限定词)
